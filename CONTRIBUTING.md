# Contributing to Alloc

Thanks for your interest in Alloc. We're not accepting pull requests at this time, but we welcome bug reports and feature requests via [GitHub Issues](https://github.com/alloc-labs/alloc/issues).

## Reporting Bugs

Open an issue with:
- What you ran (command + flags)
- What you expected
- What happened instead
- Python version, OS, GPU (if relevant)

## Feature Requests

Open an issue describing the use case. We prioritize features that help ML engineers reduce GPU costs with minimal friction.

## Security

If you discover a security vulnerability, please email **security@alloclabs.com** instead of opening a public issue.
